My Project Idea

Overview

Ideally, I would love to express my inner interest of creativity, by creating a game with the Unity Engine. This game will be a basic 2D game, with some basic graphics. I may have little experience in game making but however would still love to learn/do something new and completely different from what I'm used to.

Tools and Technologies

Unity's platform support is highly compatible, extended up to 27 platforms. The application developed can therefore be ran from PC's to Mobile devices providing for a robust and flexible experience.
